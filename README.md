# sails-hook-transformer

Transformer hook for [Sails JS](https://sailsjs.org)

### Installation

`npm install sails-hook-transformer`

### Usage

`sails.hooks.transformer([Model]);`

#### Example

```javascript
// [your-sails-app]/config/transformer.js

module.exports.transformer = {};
```
